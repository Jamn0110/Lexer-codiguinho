Codiguinho

Documentación de Codiguinho

  • Palabras clave de tipos de datos:

    • intnho es la equivalencia a int.
    • floatinho es la equivalencia a float.
    • boolinho es la equivalencia a bool.
    • boolinho es la equivalencia a bool.
    • stringho es la equivalencia a string.
    • verda es true.
    • falso es false.

  • Operadores matemáticos:

    • + -> suma
    • - -> resta
    • * -> multiplicación
    • / -> división
    • % -> modulo

  • Palabras clave de control de flujo:

    • Imprime es print.
    • ifisinho es la equivalencia a un if.
    • elifsinho es la equivalencia elif.
    • elsinho es la equivalencia a else.
    • darMole es la equivalencia a un while.
    • daStop es la equivalencia a un break.
    • volte es la esquivalencia return.

  • Símbolos de control:

    • $ define el inicio de la declaración de una variable.
    • ; define un salto de línea.
    • # define el inicio de una función.
    • = define una asignación.
    • “ ” delimitan cadenas de caracteres (strings).
    • {} delimitan bloques de código.
